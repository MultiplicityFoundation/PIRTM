# PIRTM v0.1.1

## Release Decision

**GO** (approved) for release based on current gate evidence.

## Highlights

- Clarified certificate API surface:
  - `contraction_certificate(...)` as the primary `Certificate` path
  - `ace_certificate(...)` retained for ACE-native diagnostics
- Added/expanded conformance coverage for:
  - §5 certificate types
  - §8 emission-gate policy behavior
  - §9 dual-hash witness behavior
  - §11 L0 invariants
- Added documentation and examples:
  - `docs/api/certify.md`
  - `docs/conformance.md`
  - `examples/certificate_usage.py`
