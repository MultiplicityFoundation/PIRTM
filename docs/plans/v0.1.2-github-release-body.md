# PIRTM v0.1.2

## Release Decision

**GO** (approved) for release based on validated gate evidence.

## Why v0.1.2

This patch release supersedes the prior immutable release object flow by
publishing a new tag/release with assets attached at creation time.

## Highlights

- Certificate API clarification retained:
  - `contraction_certificate(...)` as primary `Certificate` path
  - `ace_certificate(...)` for ACE-native diagnostics
- Conformance additions retained for:
  - §5 certificate types
  - §8 emission-gate behavior
  - §9 dual-hash witness behavior
  - §11 L0 invariants
- Documentation additions retained:
  - `docs/api/certify.md`
  - `docs/conformance.md`
  - `examples/certificate_usage.py`
